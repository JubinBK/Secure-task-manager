Project Overview

This project demonstrates the OWASP Top 10 vulnerabilities and their mitigation using Django.

Vulnerabilities Demonstrated:

  • SQL Injection: Login bypass & data dumping
  • Broken Access Control: IDOR in task viewing
  • Cross-Site Scripting (XSS): Stored XSS in comments
  • Security Misconfiguration: Insecure settings

Security Features Implemented:

  • Django ORM: Parameterized queries
  • Object-Level Permissions: User ownership checks
  • Template Auto-Escaping: XSS prevention
  • Secure Headers & CSRF: Built-in protection
